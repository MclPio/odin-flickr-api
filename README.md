## Installation

clone repository
```
git clone git@github.com:MclPio/odin-flickr-api.git
```

install gems
```
bundle install
```
* Gems used:
  1. [flickr](https://github.com/cyclotron3k/flickr)
  2. [figaro](https://github.com/laserlemon/figaro)

## Usage

start server
```
rails s
```

open in browser
```
localhost:3000
```

Enter a flickr user id and search
